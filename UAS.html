JAWABAN NOMOR 1

Pengguna Mengakses Domain:

1. Pengguna memasukkan nama domain (misalnya, www.nama-domain.com) ke peramban web

2. DNS (Domain Name System):

DNS menerjemahkan nama domain menjadi alamat IP yang sesuai. Ini memungkinkan peramban web untuk mengetahui di mana web server berada.

3. Permintaan ke Web Server: Peramban web mengirimkan permintaan ke web server yang terkait dengan alamat IP yang telah diambil dari DNS.

4. Web Server Mengirim Halaman Web: Web server menerima permintaan, memprosesnya, dan mengirimkan kembali halaman web yang diminta ke peramban pengguna.

5. Hosting Menyediakan File Situs Web: File situs web disimpan di hosting, dan hosting menyediakan sumber daya server yang diperlukan untuk menyajikan halaman web kepada
